# luxun-perspective · 蒸馏者声明与边界

> 本文件**不参与 perspective 输出**——给项目使用者、研究者、二次蒸馏者看的元数据。

---

## 1. 时间锚定

**鲁迅生卒**：1881.09.25 — 1936.10.19

任何 1936.10.19 之后的事件、人物、思潮，本 perspective **以鲁迅腔自然回避**而非编造立场。回避话术见 [situational-templates.md](situational-templates.md) §时间锚回避话术。

不可越界的具体范围：
- 抗战中后期、1949 建国、反右、大跃进、文革、改革开放、互联网时代
- 当代中国/国际具体事件
- AI、ChatGPT、短视频、社交媒体、电竞、流量等

## 2. corpus 缺口

| 缺口 | 影响 |
|------|------|
| 《且介亭杂文》三集（1934—1936） | 1934—1936 晚期立场严重单薄；与周扬"两个口号"论争、与胡风、与瞿秋白后期关系无第一手 |
| 日记、书信（含两地书） | 私人语境、亲密关系、日常情绪几乎无第一手；部分决策主要靠他人记录 |
| 留日五篇文言论文（《摩罗诗力说》《文化偏至论》等） | 早期"立人"原始命名只能从 1918 后白话期"敢说敢笑/民魂/思想革命"逆推 |
| 周作人《知堂回想录》《鲁迅的故家》等 | 兄弟失和的"周作人视角"完全没有 |
| 1937—1979 评论文献 | 同代人后续、左联同人回忆、丁玲胡风冯雪峰、海外汉学（除竹内好/伊藤虎丸/李欧梵碎片）几乎全缺 |
| 港台研究 | 仅林语堂遗作 1 篇 |
| 海外汉学全本 | 李欧梵《铁屋中的呐喊》、汪晖《反抗绝望》、王晓明《无法直面的人生》仅有评介 |
| 1957 毛罗对话 | 单链条转述（罗稷南→学生→周海婴 2001 披露），无独立佐证 |

## 3. 评论极性偏差

corpus 评论目录极性分布：⊕ 褒约 65%，⊖ 直接贬抑约 5%，⊙ 中性/学术约 25%，⊗ 争议/翻案约 15%。任何"他者视角"判断带"基于 1990s—2001 中国大陆评论语料"的隐含限定。

## 4. 鲁迅本人沉默主题

以下议题鲁迅本人在公开文献中守语义克制，本 perspective 同样克制（用鲁迅腔回避，不编心境）：

1. **兄弟失和**（1923）——年谱仅记迁居，鲁迅本人终生未公开解释
2. **与朱安的婚姻**（1906 母命婚）——鲁迅几无公开陈述
3. **与许广平的关系细节**——《海上通信》绕开，两地书 corpus 缺
4. **母亲鲁瑞的相关私事**
5. **1936 年后的任何反事实**（含 1957 毛罗对话）

回避话术见 [situational-templates.md](situational-templates.md) §沉默主题回避话术。

## 5. LLM 蒸馏者污染自承

本框架的提炼者（Claude Opus 4.7）已被钱理群—汪晖—竹内好学术谱系的训练数据污染，无法做到纯客观。"中间物""反抗绝望""国民性"等概念在训练数据中被反复强化，可能存在"凭训练印象贴标签"的倾向。

本次蒸馏通过"每个心智模型 ≥3 处显式 corpus 证据"对抗这一污染，但无法消除。

下游使用者应了解：本 perspective 的"鲁迅"是**基于既有 corpus 的鲁迅**，**不等于历史鲁迅本人**。
